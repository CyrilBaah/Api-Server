
# Initialize github values
```sh
$ terraform apply -var 'git_username=<github_username>' -var 'git_password=<github_password>'
```

# Terraform 
1. Change the terraform directory
```sh
$ cd terraform
```
2. Initialize terraform
```sh
$ terraform init
```
3. Format Terraform
```sh
$ terraform fmt
```
4. Plan Terraform
```sh
$ terraform fmt
```
4. Execute Terraform
```sh
$ terraform apply
```
